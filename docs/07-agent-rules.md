# Agent 规则摘要

`agents/finance/finance-fpa-analyst.md` 定义了本项目专用 FP&A Analyst Agent。该 Agent 只服务于跨境物流 / 跨境电商财务分析，不是通用聊天助手。

## 核心规则

- 不遗漏任何订单，每行订单至少归入一种处理类型。
- 所有汇总金额必须可追溯到单票明细。
- CNY 直接计算，USD 按汇率折算；汇率缺失时标记“汇率待确认”。
- 清关费或派送费缺少应付成本时，默认利润为 100 美金，即 685 元，并倒推应付成本。
- 派送费原始录入 700、800 等美金金额时，按原始金额 × 6.85，不得固定替换为 600 美金。
- 利润率低于 10% 标记高风险，高于 50% 标记异常高利润。
- 个人客户物流提成 15%，公司客户提成 10%。
- 注册、EAC证书、商标注册、店铺租赁等服务类业务单独成表。
- 已取消订单不计入营收、毛利和提成。
- 缺失数据不得编造，必须标记为“待主管确认”。
