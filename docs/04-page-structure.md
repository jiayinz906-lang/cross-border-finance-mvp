# 前端页面结构

前端使用 React + TypeScript + Vite + Ant Design，采用后台管理布局。

## 左侧菜单

1. 财务首页：月度关键指标和后端健康检查。
2. 财务台账：订单财务明细表。
3. 应收管理：应收、已回款和状态。
4. 应付管理：应付、已付款和状态。
5. 毛利分析：复杂计算规则占位。
6. 业务员提成：提成明细。
7. 异常风险：风险记录和建议。
8. 月度报表：管理层报表聚合数据。
9. Agent规则：FP&A Agent 配置和核心口径。
10. 系统设置：环境变量和云端预留说明。

## 路由

`/dashboard`、`/finance-ledger`、`/receivables`、`/payables`、`/profit-analysis`、`/commission`、`/risks`、`/reports`、`/agent-rules`、`/settings`。
