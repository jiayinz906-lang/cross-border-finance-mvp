# 财务计算规则

1. CNY / RMB / 人民币 / 元直接按人民币计算。
2. USD / 美金 / 美元 / $ 需要折算成人民币。
3. 汇率优先使用 Excel 原始值。
4. 汇率缺失时调用 ExchangeRate-API。
5. ExchangeRate-API Key 使用 `EXCHANGE_RATE_API_KEY` 环境变量。
6. 清关费或派送费缺少应付时，默认利润为 100 美金，即 685 元。
7. 倒推应付成本 = 应收费用折人民币金额 - 685。
8. 派送费如原始数据已录入 700、800 等美金金额，必须按原始金额 × 6.85。
9. 利润率 < 10% 标记为高风险。
10. 利润率 > 50% 标记为异常高利润。
11. 物流业务个人客户提成 15%。
12. 公司客户提成 10%。
13. 注册、EAC证书、商标注册、店铺租赁等服务类业务单独成表。
14. 已取消订单不计入营收、毛利和提成。

## 核心公式

应收总额 = 应收物流费 + 应收清关费 + 应收派送费 + 应收其他费用。

应付总额 = 应付物流成本 + 应付清关成本 + 应付派送成本 + 应付其他成本。

调整后毛利 = 调整后应收总额 - 调整后应付总额。

调整后毛利率 = 调整后毛利 / 调整后应收总额；应收总额为 0 时标记为“无法计算”。
