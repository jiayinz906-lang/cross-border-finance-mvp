# 财务计算规则

## 汇率规则

1. 不使用外部汇率 API。
2. 严格按原始 Excel 表格标注汇率计算。
3. 汇率标注为 `1` 时，按人民币计算。
4. 汇率标注为 `美金`、`美元`、`USD`、`$`、`美金汇率未出`、`汇率未出` 时，统一按 `6.85` 计算。
5. 汇率标注为其他数字时，严格按该数字计算。
6. 汇率无法识别时，标记为“待主管确认”。

## 应收应付

应收总额 = 应收运费 + 应收清关/报关费 + 应收派送费 + 其他应收。

应付总额 = 应付运费 + 应付清关/报关成本 + 应付派送成本 + 其他应付。

调整后毛利 = 调整后应收总额 - 调整后应付总额。

调整后毛利率 = 调整后毛利 / 调整后应收总额；应收总额为 0 时标记为“无法计算”。

## 成本补录

1. 清关费或派送费缺少应付时，默认利润为 100 美金，即 685 元。
2. 倒推应付成本 = 应收费用折人民币金额 - 685。
3. 倒推成本小于 0 时，标记为“待主管确认”。
4. 派送费如原始数据已录入 700、800 等美金金额，必须按原始金额 × 6.85，不得固定替换为 600 美金。

## 风险与提成

1. 利润率低于 10% 标记为高风险。
2. 利润率高于 50% 标记为异常高利润。
3. 物流业务个人客户提成按月度物流毛利阶梯计算。
4. 公司客户提成 10%。
5. 注册、EAC 证书、商标注册、店铺租赁等服务类业务单独成表。
6. 已取消订单不计入营收、毛利和提成。
